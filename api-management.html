How to Use Your API Keys

Follow these instructions to use your API keys with the Robotics AI Cells API:

1. Authentication

Include your API key in the request headers:

X-API-Key: your_api_key_here

2. Making API Requests

Send a POST request to the AI Cells endpoint:

POST https://robotics-ai-cells.onrender.com/ai-api

3. Request Format

Format your request body as follows:

{ "cells": ["cell_id"], "data": { "cell_id": { "param1": value1, "param2": value2 } } }

4. Example (JavaScript)

const apiKey = "your_api_key_here"; const apiUrl = "https://robotics-ai-cells.onrender.com/ai-api"; fetch(apiUrl, { method: 'POST', headers: { 'Content-Type': 'application/json', 'X-API-Key': apiKey }, body: JSON.stringify({ cells: ["fast_math"], data: { "fast_math": { "angle": 45 } } }) }) .then(response => response.json()) .then(data => console.log(data)) .catch(error => console.error('Error:', error));
