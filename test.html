Interactive Neural Networks Demo

Feedforward Neural Network (FNN)

A simple network where data flows in one direction from input to output. Used for tasks like digit classification.

Convolutional Neural Network (CNN)

Designed for images, using convolutional layers to detect features like edges. Common in image recognition.

Recurrent Neural Network (RNN)

Handles sequential data with loops to retain memory of previous inputs. Used for time-series or text.

Long Short-Term Memory (LSTM)

An RNN variant that remembers longer sequences using gates. Great for language modeling.

Generative Adversarial Network (GAN)

Two networks (generator and discriminator) compete to create realistic data, like images.

Transformer

Uses attention mechanisms to focus on important parts of input. Powers modern NLP like chatbots.
