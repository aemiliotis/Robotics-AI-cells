DSKY SIMULATOR INSTRUCTIONS

  • VERB/NOUN: Enter command codes (e.g. Verb 06 Noun 68 for time)
  • ENTR: Confirm entry
  • CLR: Clear current input
  • RSET: Reset system
  • PRO: Trigger program alarm
  • KEY REL: Toggle key release light

Try: Verb 37 Noun 01 (Engine status) or Verb 06 Noun 68 (Mission time)
