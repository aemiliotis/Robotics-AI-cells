Προσομοίωση Οπτικοποίησης Embedding Layers του TinyLlama

Αυτή η σελίδα προσομοιώνει την οπτικοποίηση των embeddings από το μοντέλο TinyLlama. Εισάγετε κείμενο, και θα εμφανιστούν τυχαία δημιουργημένα 2D σημεία (προσομοίωση t-SNE) για κάθε token. Στην πραγματικότητα, θα χρησιμοποιούσαμε Python για εξαγωγή πραγματικών embeddings, αλλά εδώ χρησιμοποιούμε JavaScript για προσομοίωση.
